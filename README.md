# Morphological Productivity of LLM Italian Neologisms in a Reverse Dictionary Task

Course project for Computational Linguistics, University of Trento – CIMeC.

This repository contains the data, scripts, and analysis files used to study the morphological productivity of Italian neologisms generated by large language models (LLMs), compared against human-coined and randomly-generated baselines, using a Reverse Dictionary Task and human plausibility ratings.

**Authors:** Kaitlin Zahendra, Giohanna Pennella, Mattia Bressan
`{kaitlin.zahendra, giohanna.pennella, mattia.bressan}@studenti.unitn.it`

## Overview

The project investigates whether human plausibility judgments of Italian neologisms correlate with the productivity of the morphological word-formation strategies used to create them. Candidate words were generated from four sources — human speakers, two LLMs (Gemma 4 e4b and Gemini 2.5 Flash), and a random bigram baseline — for a shared set of definitions drawn from *Slengo.it*, a crowdsourced repository of contemporary Italian slang. Generated words were rated for plausibility by native Italian speakers via an online survey, and the results were analyzed both statistically (by word source) and morphologically (by word-formation pattern).

## Repository Structure

```
.
├── slengo_scraper.py                  # Scrapes term/definition data from slengo.it
├── gemini_creation_words.py           # Generates candidate neologisms via the Gemini 2.5 Flash API
├── RandoWordGen                       # Generates random baseline words via bigram concatenation
├── Definitions and Words.xlsx         # Target definitions and the generated words per source
├── Morphological Analysis.xlsx        # Morphological annotation and pattern-frequency analysis
└── Survey results + Analysis.xlsx     # Raw survey responses and statistical analysis (ANOVA, plots)
```

### Scripts

- **`slengo_scraper.py`** — Scrapes the *Slengo.it* index of terms (`/browse`) and downloads each term's definition and usage examples. Outputs results to `slengo_dictionary.csv` / `slengo_dictionary.json`, with checkpointing to resume interrupted runs.
  Dependencies: `requests`, `beautifulsoup4`.

- **`gemini_creation_words.py`** — Calls the Gemini 2.5 Flash API to generate a novel Italian word for a given definition, using a fixed prompt and generation parameters (temperature, top-k, top-p) for reproducibility.
  Requires a `GEMINI_API_KEY` environment variable. Dependencies: `requests`.

- **`RandoWordGen`** — Generates phonotactic baseline words by segmenting existing Italian words into overlapping bigrams and randomly concatenating a sample of them. Used to produce the lower-baseline ("Random") condition in the experiment.
  Dependencies: standard library only (`random`).

### Data files

- **`Definitions and Words.xlsx`** — The 27 target definitions used as prompts, alongside the corresponding generated word for each of the four sources (Human, Gemma 4 e4b, Gemini 2.5 Flash, Random).

- **`Morphological Analysis.xlsx`** — Morphological annotation of the curated Slengo noun subset and of the generated words, including sheets for general statistics, per-category morphological stats, pattern frequency, error analysis (ungrammatical/spurious LLM generations), and comparison against the Slengo corpus distribution.

- **`Survey results + Analysis.xlsx`** — Raw plausibility ratings collected from the Reverse Dictionary Task survey (per word and per control item), plus the statistical analysis (ANOVA, post-hoc tests, box plots) comparing plausibility across word sources.

## Reproducing the Pipeline

1. **Scrape source data**: run `slengo_scraper.py` to collect Italian neologisms and definitions from Slengo.it.
2. **Curate definitions**: manually select and annotate relevant noun entries (see `Morphological Analysis.xlsx`) to identify 27 target definitions.
3. **Generate candidate words**:
   - Human: produced manually by the authors.
   - LLM: run `gemini_creation_words.py` (and the equivalent Gemma 4 e4b API call) for each definition.
   - Random baseline: run `RandoWordGen`.
4. **Run the survey**: present each definition with its four candidate words (randomized order) and collect plausibility ratings (1–5 Likert scale) from native Italian speakers.
5. **Analyze results**: see `Survey results + Analysis.xlsx` for statistical comparisons across sources, and `Morphological Analysis.xlsx` for the morphological breakdown of generated words.

## Requirements

- Python 3.9+
- `requests`
- `beautifulsoup4`
- A Gemini API key (set as an environment variable, e.g. `GEMINI_API_KEY`) for `gemini_creation_words.py`

Install dependencies with:

```bash
pip install requests beautifulsoup4
```
